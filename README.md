# Film Guesser

## Quickstart

Unix: `.\mvnw javafx:run`  
Windows: `./mvnw.cmd javafx:run`  

## Description

Guess films.

## Architecture

Entry point: `Main.java`

### Four main components:
- State `State.java`
- User interfaces `ui`
- Network services `services`
- Logic (`Main.java` for now but could make dedicated logic folder)
